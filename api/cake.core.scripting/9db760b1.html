ScriptProcessorContext Class

  1. API Documentation
  2. Cake.Core.Scripting
  3. ScriptProcessorContext
  • System.Object
    • ScriptProcessorContext
Acts as a context, keeping track of loaded scripts, references and namespaces to add and script content.
Constructors
Name Description
ScriptProcessorContext() Initializes a new instance of the ScriptProcessorContext class.
Properties
Name Type Description
References HashSet<String> Gets the script's assembly references referenced via the #r directive.
Namespaces HashSet<String> Gets the script's namespaces imported via the using statement.
ProcessedScripts HashSet<String> Gets the scripts that has been processed so far.
Methods
Name Description
HasScriptBeenProcessed(String) Determines whether the specified script has been processed.
MarkScriptAsProcessed(String) Marks the script as processed.
AddReference(String) Adds an assembly reference to be loaded later.
AddNamespace(String) Adds a namespace to be loaded later.
AddScriptAliasCode(String) Adds the code for a script alias.
AppendScriptLine(String) Appends a line to the script.
GetScriptCode() Gets the script code.
