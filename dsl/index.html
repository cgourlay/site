Here you can find information about Cake alias methods and properties.

Arguments
HasArgument(string) Method Determines whether or not the specified argument exist.

Assembly Info

  • Clean
CleanDirectories(string) Method Cleans the directories matching the specified pattern. Cleaning the directory will remove all it's content but not the directory itself.

CleanDirectory(DirectoryPath) Method Cleans the specified directory.
CleanDirectory(DirectoryPath, Func<IFileSystemInfo, bool>) Method Cleans the specified directory.
  • Copy
CopyDirectory(DirectoryPath, DirectoryPath) Method Copies a directory to the specified location.
  • Create
CreateDirectory(DirectoryPath) Method Creates the specified directory.
  • Delete
DeleteDirectories(IEnumerable<DirectoryPath>, bool) Method Deletes the specified directories.
DeleteDirectories(IEnumerable<string>, bool) Method Deletes the specified directories.
DeleteDirectory(DirectoryPath, bool) Method Deletes the specified directory.
  • Exists
DirectoryExists(DirectoryPath) Method Determines whether the given path refers to an existing directory.
Environment
EnvironmentVariable(string) Method Retrieves the value of the environment variable or null if the environment variable do not exist.
HasEnvironmentVariable(string) Method Checks for the existence of a value for a given environment variable.

  • Copy

CopyFiles(IEnumerable<FilePath>, DirectoryPath) Method Copies existing files to a new location.
CopyFiles(IEnumerable<string>, DirectoryPath) Method Copies existing files to a new location.
  • Delete
DeleteFiles(string) Method Deletes the specified files.
DeleteFiles(IEnumerable<FilePath>) Method Deletes the specified files.
DeleteFile(FilePath) Method Deletes the specified file.
  • Exists
FileExists(FilePath) Method Determines whether the given path refers to an existing file.
  • Move
MoveFileToDirectory(FilePath, DirectoryPath) Method Moves an existing file to a new location.
MoveFiles(string, DirectoryPath) Method Moves existing files matching the specified pattern to a new location.
MoveFiles(IEnumerable<FilePath>, DirectoryPath) Method Moves existing files to a new location.
MoveFile(FilePath, FilePath) Method Moves an existing file to a new location, providing the option to specify a new file name.

NuGetHasSource(string) Method Checks whether or not a NuGet package source exists in the global user configuration, using the specified source.

NuGetInstallFromConfig(FilePath, NuGetInstallSettings) Method Installs NuGet packages using the specified package configuration and settings.
  • Pack
NuGetPack(FilePath, NuGetPackSettings) Method Creates a NuGet package using the specified Nuspec file.
  • Push

  • RemoveSource
NuGetRemoveSource(string, string) Method Removes NuGet package source using the specified name & source from global user config

  • Restore
NuGetRestore(FilePath) Method Restores NuGet packages for the specified target.
NuGetRestore(FilePath, NuGetRestoreSettings) Method Restores NuGet packages using the specified settings.
  • SetApiKey
NuGetSetApiKey(string, string, NuGetSetApiKeySettings) Method Installs NuGet packages using the specified API key, source and settings.
NuGetSetApiKey(string, string) Method Installs NuGet packages using the specified API key and source.
