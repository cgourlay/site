NuGetAliases.NuGetPush Method

  1. API Documentation
  2. Cake.Common.Tools.NuGet
  3. NuGetAliases
Pushes a NuGet package to a NuGet server and publishes it.
Parameters
  • context
    • Type: ICakeContext
    • The context.
  • packageFilePath
    • Type: FilePath
    • The .nupkg file path.
  • settings
    • Type: NuGetPushSettings
    • The settings.
Return value
No return value.
Example
// Get the path to the package.
var package = "./nuget/SlackPRTGCommander.0.0.1.nupkg";
            
// Push the package.
NuGetPush(package, new NuGetPushSettings {
    Source = "http://example.com/nugetfeed",
    ApiKey = "4003d786-cc37-4004-bfdf-c4f3e8ef9b3a"
});
