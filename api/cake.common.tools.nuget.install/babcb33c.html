NuGetInstallSettings Class

  1. API Documentation
  2. Cake.Common.Tools.NuGet.Install
  3. NuGetInstallSettings

    • NuGetInstallSettings
Contains settings used by NuGetInstaller.
Constructors
Name Description
NuGetInstallSettings()
Properties
Name Type Description
ToolPath FilePath Path to nuget.exe
OutputDirectory DirectoryPath Specifies the directory in which packages will be installed. If none specified, uses the current directory.
Version String The version of the package to install. If none specified, uses the latest.
ExcludeVersion Boolean If set, the destination folder will contain only the package name, not the version number
Prerelease Boolean Allows prerelease packages to be installed. This flag is not required when restoring packages by installing from packages.config.
RequireConsent Boolean Checks if package install consent is granted before installing a package.
SolutionDirectory DirectoryPath Solution root for package restore.
Source ICollection<String> A list of packages sources to use for this command.
NoCache Boolean Sets whether or not to use the machine cache as the first package source.
DisableParallelProcessing Boolean Disable parallel processing of packages for this command.
Verbosity Nullable<NuGetVerbosity> Display this amount of details in the output: Normal, Quiet, Detailed.
ConfigFile FilePath The NuGet configuration file. If not specified, file %AppData%\NuGet\NuGet.config is used as configuration file.
