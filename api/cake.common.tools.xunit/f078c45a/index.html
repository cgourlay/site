XUnit2Settings Class

  1. API reference
  2. Cake.Common.Tools.XUnit
  3. XUnit2Settings

Contains settings used by XUnit2Runner.

Syntax
public class XUnit2Settings
Constructors
Name Description
XUnit2Settings(​)​ Initializes a new instance of the XUnit2Settings class.
Properties
Name Description
HtmlReport Gets or sets a value indicating whether an HTML report should be generated.
OutputDirectory Gets or sets the output directory.
ShadowCopy Gets or sets a value indicating whether tests should be run as a shadow copy. Default value is true.
ToolPath Gets or sets the tool path.
XmlReport Gets or sets a value indicating whether an XML report should be generated.
XmlReportV1 Gets or sets a value indicating whether an xUnit.net v1 style XML report should be generated.
