NUnitSettings Class

  1. API reference
  2. Cake.Common.Tools.NUnit
  3. NUnitSettings

Contains settings used by NUnitRunner.

Syntax
public sealed class NUnitSettings
Constructors
Name Description
NUnitSettings() Initializes a new instance of the NUnitSettings class.
Properties
Name Description
Exclude Gets or sets the categories to exclude from a run.
Framework Gets or sets the version of the runtime to be used when executing tests.
Include Gets or sets the categories to include in a run.
NoLogo Gets or sets a value indicating whether to show copyright information at the start of the program.
NoResults Gets or sets a value indicating whether to generate the XML result file.
NoThread Gets or sets a value indicating whether the main thread should be used for running tests.
ResultsFile Gets or sets the name of the XML result file.
ShadowCopy Gets or sets a value indicating whether tests should be run as a shadow copy. Default value is true.
StopOnError Gets or sets a value indicating whether execution of the test run should terminate immediately on the first test failure or error.
Timeout Gets or sets the default timeout to be used for test cases in this run. If any test exceeds the timeout value, it is cancelled and reported as an error.
ToolPath Gets or sets the tool path.
Trace Gets or sets the amount of information that NUnit should write to its internal trace log.
