NUnitSettings Class

  1. API Documentation
  2. Cake.Common.Tools.NUnit
  3. NUnitSettings
  • System.Object
    • NUnitSettings
Contains settings used by NUnitRunner.
Constructors
Name Description
NUnitSettings() Initializes a new instance of the NUnitSettings class.
Properties
Name Type Description
ToolPath FilePath Gets or sets the tool path.
ResultsFile FilePath Specifies the name of the Xml Results File.
NoResults Boolean Specifies wether to generate the results xml file.
Framework String Allows you to specify the version of the runtime to be used in executing tests.
Include String Specifies categories to include in running. See http://nunit.org/index.php?p=consoleCommandLine&r=2.6.3 under "Specifying Test Categories to Include or Exclude".
Exclude String Specifies categories to exclude from running. See http://nunit.org/index.php?p=consoleCommandLine&r=2.6.3 under "Specifying Test Categories to Include or Exclude".
Timeout Nullable<Int32> Represents the default timeout to be used for test cases in this run. If any test exceeds the timeout value, it is cancelled and reported as an error.
ShadowCopy Boolean Gets or sets a value indicating whether tests should be run as a shadow copy. Default value is true .
NoThread Boolean Suppresses use of a separate thread for running the tests and uses the main thread instead.
NoLogo Boolean Disables display of the copyright information at the start of the program.
StopOnError Boolean Causes execution of the test run to terminate immediately on the first test failure or error.
Trace String Allows you to control the amount of information that NUnit writes to its internal trace log.
