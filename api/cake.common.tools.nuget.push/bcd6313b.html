Contains settings used by NuGetPusher.
Constructors
Name Description
NuGetPushSettings()
Properties
Name Type Description
Source String Gets or sets the server URL. If not specified, nuget.org is used unless DefaultPushSource config value is set in the NuGet config file. Starting with NuGet 2.5, if NuGet.exe identifies a UNC/folder source, it will perform the file copy to the source.
ApiKey String Gets or sets the API key for the server.
Timeout Nullable<TimeSpan> Gets or sets the timeout for pushing to a server. Defaults to 300 seconds (5 minutes).
Verbosity Nullable<NuGetVerbosity> Gets or sets the verbosity.
ConfigFile FilePath Gets or sets the NuGet configuration file.
ToolPath FilePath Gets or sets the tool path.
