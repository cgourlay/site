NuGet

If an addin exists on NuGet, then it can be included in your script via the #addin directive. Any NuGet package can be imported with the #addin directive.

Usage example

This usage example will import the addin Cake.Slack directly from NuGet.

#addin "Cake.Slack"

var token = EnvironmentVariable("SLACK_TOKEN");
var channel = "#cake";
var text = "This _is_ a `message` from *CakeBuild*";

// Post the message.
var result = SlackChatPostMessage(token, channel, text);

if (result.Ok)
{
    // Everything went fine!
    Information("Message was succcessfully sent.");
}
else
{
    // An error occured.
    Error("Failed to send message: {0}", result.Error);
}
