RequireConsent Property

  1. API reference
  2. Cake.Common.Tools.NuGet.Restore
  3. NuGetRestoreSettings
  4. RequireConsent

Gets or sets a value indicating whether package restore consent is granted before installing a package.

Property Value
Type: bool

Value: true if package restore consent is granted; otherwise, false.
