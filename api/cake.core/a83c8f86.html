CakeTask Class

  1. API Documentation
  2. Cake.Core
  3. CakeTask
  • System.Object
    • CakeTask
A CakeTask represents a unit of work.
Properties
Name Type Description
Name String Gets the name of the task.
Description String Gets or sets the description of the task.
Dependencies IReadOnlyList<String> Gets the task's dependencies.
Criterias IReadOnlyList<Func<Boolean>> Gets the task's criterias.
ErrorHandler Action<Exception> Gets the error handler.
ErrorReporter Action<Exception> Gets the error reporter.
FinallyHandler Action Gets the finally handler.
Methods
Name Description
AddDependency(String) Adds a dependency to the task.
AddCriteria(Func<Boolean>) Adds a criteria to the task that is invoked when the task is invoked.
SetErrorHandler(Action<Exception>) Sets the error handler for the task. The error handler is invoked when an exception is thrown from the task.
SetErrorReporter(Action<Exception>) Sets the error reporter for the task. The error reporter is invoked when an exception is thrown from the task. This action is invoked before the error handler, but gives no opportunity to recover from the error.
SetFinallyHandler(Action) Sets the finally handler for the task. The finally handler is always invoked when a task have finished running.
Execute(ICakeContext) Executes the task using the specified context.
