Error(ICakeLog, Verbosity, string, Object[]) ExtensionMethod

  1. API reference
  2. Cake.Core.Diagnostics
  3. LogExtensions
  4. Error(ICakeLog, Verbosity, string, Object[])

Writes an error message to the log using the specified verbosity and format information.

Syntax
public static void Error(
       this ICakeLog log,
       Verbosity verbosity,
       string format,
       Object[] args
)
Parameters
log
Type: ICakeLog
The log.
verbosity
Type: Verbosity
The verbosity.
format
Type: string
A composite format string.
args
Type: Object[]
An array of objects to write using format.
