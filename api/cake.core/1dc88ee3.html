CakeTaskBuilderExtensions Class

  1. API Documentation
  2. Cake.Core
  3. CakeTaskBuilderExtensions
  • System.Object
    • CakeTaskBuilderExtensions
Contains extension methods for CakeTaskBuilder<T>.
Methods
Name Description
IsDependentOn<T>(CakeTaskBuilder<T>, String) Creates a dependency between two tasks.
WithCriteria<T>(CakeTaskBuilder<T>, Boolean) Adds a criteria that has to be fulfilled for the task to run.
WithCriteria<T>(CakeTaskBuilder<T>, Func<Boolean>) Adds a criteria that has to be fulfilled for the task to run. The criteria is evaluated when traversal of the graph occurs.
Does(CakeTaskBuilder<ActionTask>, Action) Adds an action to be executed when the task is invoked.
Does(CakeTaskBuilder<ActionTask>, Action<ICakeContext>) Adds an action to be executed when the task is invoked.
Description<T>(CakeTaskBuilder<T>, String) Adds a description to the task.
ContinueOnError(CakeTaskBuilder<ActionTask>) Adds an indication to the task that a thrown exception will not halt the script execution.
OnError<T>(CakeTaskBuilder<T>, Action) Adds an error handler to be executed if an exception occurs in the task.
OnError<T>(CakeTaskBuilder<T>, Action<Exception>) Adds an error handler to be executed if an exception occurs in the task.
Finally<T>(CakeTaskBuilder<T>, Action) Adds a finally handler to be executed after the task have finished executing.
ReportError<T>(CakeTaskBuilder<T>, Action<Exception>) Adds an error reporter for the task to be executed when an exception is thrown from the task. This action is invoked before the error handler, but gives no opportunity to recover from the error.
